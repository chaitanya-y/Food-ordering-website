Intro to AnoClock

AnoClock is a jQuery plug-in that allows you to simply embed anolog clocks into your project. It calculates the angles of the second, minutes and hour hands based off of the set time. AnoClock lets you customize the arm movements, adjust timezones and set static clocks. It also lets you simply and quickly customize the styling to match your project. The plug-in can be used on existing clock UI's or by using the default clock UI provided.

Getting started

To get started, AnoClock needs a container with a clock object inside.

HTML:

<div class='clock-container'>
 <div class='clock'></div>
</div>

The container object needs to have a set width and height with square dimensions.

CSS:

.clock-container {
 width: 200px;
 height: 200px;
}

Lastly, you need to initialize the clock object with the follow command.

JS

$('.clock').setAsClock();

There are a few simple UI settings that makes adjusting this clock quick and easy to do.

$('.clock').setAsClock({
 sweeping: false, //Changes the hands to a ticking
 numbers: true, //false will turn the numbers off
 numberSize: '24px', //Adjust the font size of the numbers
 ticks: 1, //0-2 changes the tick count
 backgroundColor: '#500', //can take any normal css color
 color: 'lightblue', //Changes the color of all the forground items
 borderRadius: '60px' //change how round the clock is
});

All Options

Options Type Details
initUI Boolean initUI will render the DOM objects necesary to build a clock. It paints the second, minute, and hour hands. Default is set to true.
styling Boolean Sets the default UI options and/or uses the options set to paint the clock. No further css will be required. Default is set to true.
sweeping Boolean Sets the motion of sweep on the hands of the clock. Setting to false will cause a ticking motion. Default is set to true
numbers Boolean Turns on and off the painting of the numbers in the UI. Default is set to true.
numberSize String Sets the size of the numbers on the clock face. Example: '18px' or '2em'. Default is set to '18px'
backgroundColor String, Hex Color, RGB Color Sets the color of the face of the watch. Can take any normal css color. Example: 'blue', #00f, rgb(0,0,255). Default is set to 'steelblue'.
ticks Number Defines the density of the tick marks to be painted. Requires 0, 1 or 2. Default is set to 2.
borderRadius String Sets the border-radius of the clock face. Requires a string of percent or pixel. Example: '100%' or '100px'. Default is set to '100%'.
color String, Hex Color, RGB Color Sets the color of the text, ticks and hands. Can take any normal css color. Example: 'blue', #00f, rgb(0,0,255). Default is set to 'white'.
UTCOffset Number Sets the timezone of the clock based off of UTC Offset. Requires a positive or negative number. be careful as different locations will change their offset depending on the time of year. Default is set to users time.

New York '-5'
